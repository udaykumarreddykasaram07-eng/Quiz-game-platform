# 🎮 Quiz Game Platform

A fully functional web-based quiz game platform built with Flask, SQLite, HTML, CSS, and JavaScript.

## 📋 Features

### User Features
- ✅ User registration and authentication
- ✅ Multiple quiz categories (8 categories)
- ✅ Interactive quiz gameplay with timer
- ✅ Real-time scoring and progress tracking
- ✅ Detailed results and performance analytics
- ✅ Global leaderboard
- ✅ User profile with statistics
- ✅ Keyboard shortcuts for navigation

### Admin Features
- ✅ User management
- ✅ Question management
- ✅ Category management
- ✅ Performance analytics
- ✅ Database statistics

### Technical Features
- ✅ Responsive design (mobile-friendly)
- ✅ Smooth animations and transitions
- ✅ REST API endpoints
- ✅ Session management
- ✅ Form validation
- ✅ Error handling
- ✅ Flash messages

## 🛠️ Technology Stack

### Backend
- **Python 3.x**
- **Flask** - Web framework
- **Flask-SQLAlchemy** - ORM
- **Flask-Bcrypt** - Password hashing
- **Flask-Login** - User authentication
- **SQLite** - Database

### Frontend
- **HTML5** - Structure
- **CSS3** - Styling with custom animations
- **JavaScript** - Interactivity
- **Font Awesome** - Icons
- **Google Fonts** - Typography
